Prettconf - simple CLI to create your prettier config. Written in TypeScript.

## Instalation

With yarn

```bash
$ yarn global add prettconf
```

With npm

```bash
$ npm install -g prettconf
```

## Usage

```bash
$ cd your_folder

$ npx prettconf
```
